目标：把“获客、转化、交付、续费”串成可执行的标准动作。下方流程可用于销售、教务、班主任和学管团队协同，帮助提升试听转化率和续课率。 Goal: Turn acquisition, conversion, delivery, and renewal into standardized and executable actions. The process below helps sales, academic operations, homeroom staff, and student management teams collaborate to improve trial-to-enrollment conversion and renewal rates.

Step 01

试课流程Trial Class Workflow

  • 线索进入：登记家长/学员基础信息与学习目标。
  • 课前诊断：能力测评 + 学情访谈，输出试听策略。
  • 试听执行：Demo教学 + 互动反馈 + 课堂亮点记录。
  • 课后转化：24小时内发送测评报告并邀约选课。
  • Lead Intake: Register parent/student profile and learning goals.
  • Pre-Class Diagnosis: Conduct ability assessment + learning interview to define trial strategy.
  • Trial Delivery: Demo teaching + interactive feedback + highlight capture.
  • Post-Class Conversion: Send assessment report and course invitation within 24 hours.

Step 02

排课（选课）流程Scheduling (Course Selection) Workflow

  • 方案确认：按学段与目标定制课程包和阶段目标。
  • 资源匹配：匹配主讲老师、时段、班型（1v1/小班）。
  • 排课锁定：确认正式课表，生成开课日历与提醒机制。
  • 服务建群：建立家校沟通群，明确周反馈节奏。
  • Plan Confirmation: Customize course package and stage goals by grade and objective.
  • Resource Matching: Assign lead teacher, timeslot, and class type (1-on-1/small group).
  • Schedule Lock-in: Finalize timetable and set class calendar with reminders.
  • Service Group Setup: Build home-school communication group with weekly feedback rhythm.
➜
Step 03

续课流程Renewal Workflow

  • 过程跟踪：每周学习报告 + 作业达成 + 阶段测评。
  • 续课预警：剩余课时 ≤ 20% 触发续课沟通。
  • 价值复盘：展示成长曲线、目标达成与下一阶段路径。
  • 续费成交：确认新课包、课表衔接与优惠政策。
  • Progress Tracking: Weekly report + homework completion + milestone assessment.
  • Renewal Alert: Trigger renewal communication when remaining class hours ≤ 20%.
  • Value Review: Present growth curve, target achievement, and next-stage roadmap.
  • Renewal Closing: Confirm new package, timetable continuity, and offer policy.
